A Test Page (Static) on Cloudflare Worker Sites

This site was built by issuing the command below. 

 wrangler generate my-app https://github.com/cloudflare/worker-sites-template
Then editing the wrangler.toml file, and the static html in the public folder.. and then 
 wrangler publish
